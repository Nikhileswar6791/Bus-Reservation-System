# Bus-Reservation-System
1. This is a platform where user can signup if he/she is a new user else he can login to see the seat availability and book/cancel tickets
2. The changes in the seat availability made by one user will be reflected for others also
3. Finally Total amount for the tickets user selected for booking is displayed
4. Admin is the one who can restart the booking, after that particular journey was ended.
